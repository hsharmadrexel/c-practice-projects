Title: Shell Project
Name: Harsh Sharma

This version of the shell supports:
1. Built-in cd command processing
2. & Standard Notation for background processing
3. Standard commands execution
4. Command sequencing with & and ;
5. Input/output redirection - truncating and appending.
6. Pipe redirection

Comments:
The shell is functional and meets all of the requirements as specified above.

Supplement Instructions:
1. Using a linux server or a machine that has the GCC compiler, compile the source code files with the command below:
   gcc -o sh sh.c

2. Now that you have the executable file created, launch the sh283 executable with the command below and enter in user input as needed:
   ./sh

3. To log out of the shell program, simply indicate EOF on the standard input which is Cntrl + D on the keyboard.
